## Install

```shell
npm install meisha-ui -S

# if you use yarn
yarn add meisha-ui
```

## Quick Start

```javascript
import Vue from 'vue';
import Meisha from 'meisha-ui';
import 'meisha-ui/lib/styles/index.css';

Vue.use(Meisha);
```

## Development

1. 请先 fork 一份到自己的项目下，不要直接在仓库下建分支。
2. 开发环境搭建
   ```shell
   git clone git@github.com:meishaFE/meisha-ui.git
   npm run dev
   ```
